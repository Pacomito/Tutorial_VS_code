# VS Code Keyboard Shortcuts for R Analysis

## Essential Shortcuts

| Action | Windows/Linux | Mac | Notes |
|--------|---------------|-----|-------|
| **Command Palette** | `Ctrl+Shift+P` | `Cmd+Shift+P` | Access any command |
| **Run R line/selection** | `Ctrl+Enter` | `Cmd+Enter` | Most used! |
| **Save file** | `Ctrl+S` | `Cmd+S` | Save often! |
| **Open terminal** | `` Ctrl+` `` | `` Cmd+` `` | R runs here |
| **New file** | `Ctrl+N` | `Cmd+N` | Create script |
| **Open file** | `Ctrl+P` | `Cmd+P` | Quick file search |

## R-Specific

| Action | Windows/Linux | Mac | Notes |
|--------|---------------|-----|-------|
| **Run source** | Right-click menu | Right-click menu | Runs entire script |
| **Run selection** | `Ctrl+Enter` | `Cmd+Enter` | Selected code only |
| **Clear console** | `Ctrl+L` in terminal | `Cmd+L` in terminal | Clean R output |

---

## Pro Tips

💡 **Command Palette is your friend**: Can't remember a shortcut? Press `Ctrl+Shift+P` and type what you want!

💡 **Customize shortcuts**: File → Preferences → Keyboard Shortcuts

💡 **Learn one at a time**: Start with `Ctrl+Enter` and `Ctrl+S`, add others gradually
